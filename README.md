





## 安装

#### 安装govendor 工具

```
go get -u -v github.com/kardianos/govendor
```

#### 支持make file的平台

```
git clone https://github.com $GOPATH/src/github.com/GM/
cd $GOPATH/src/github.com/GM/
make
```

就可以完成编译安装
如有需要欢迎联系ganmani@vip.163.com 
### 更新GM

```
make update
```

### 更新 vendor

```
make updatevendor
```
